# board-login-img-security-Oauth — 소셜 로그인

`board-login-img-security` + Google · Naver · Kakao OAuth2 (`roomfit-social` / `shopping-mall1` 패턴)

## 실행

- 포트: **8086**
- DB: `board_login_img_security_oauth`
- 업로드: `~/board-login-img-security-oauth-uploads/`

```bash
cd board-login-img-security-Oauth
./mvnw spring-boot:run
```

## Redirect URI (로컬)

| 제공자 | Redirect URI |
|--------|----------------|
| Google | `http://localhost:8086/login/oauth2/code/google` |
| Naver | `http://localhost:8086/login/oauth2/code/naver` |
| Kakao | `http://localhost:8086/login/oauth2/code/kakao` |

## 환경 변수

`application.yml` 기본값 `change-me`이면 소셜 로그인이 동작하지 않습니다.

```text
GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
NAVER_CLIENT_ID=...
NAVER_CLIENT_SECRET=...
KAKAO_CLIENT_ID=...
KAKAO_CLIENT_SECRET=   # 선택
```

Windows PowerShell 예:

```powershell
$env:GOOGLE_CLIENT_ID="..."
$env:GOOGLE_CLIENT_SECRET="..."
./mvnw spring-boot:run
```

## 동작

1. `/login` → 폼 로그인 또는 소셜 버튼
2. OAuth 콜백 → `CustomOAuth2UserService` → `UserService.registerOrUpdateOAuthUser`
3. Principal: `LoginUser` (폼·소셜 공통)
4. 게시판 권한: `SecurityUtils.currentUsername()` = DB `username`

## 주요 파일

- `security/LoginUser.java` — UserDetails + OAuth2User
- `security/CustomOAuth2UserService.java`
- `config/SecurityConfig.java` — `oauth2Login`
- `application.yml` — `spring.security.oauth2.client`
