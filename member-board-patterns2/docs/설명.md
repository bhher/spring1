# member-board-patterns2 — 3단계: Builder 패턴 (Lombok)

**Lombok `@Builder`** 로 엔티티·응답 DTO를 조립합니다. 필드가 많을 때 가독성이 좋고, 선택 필드를 다루기 쉽습니다.

## 핵심 코드 위치

| 구분 | 파일 | 내용 |
|------|------|------|
| DTO → Entity | `UserService.register` | `User.builder().email(...).passwordHash(...).build()` |
| Entity → DTO | `UserResponseDto.from`, `BoardResponseDto.from` | `UserResponseDto.builder()...build()` |
| 수정 | `Board.edit` | 영속 엔티티 변경은 Builder 대신 **도메인 메서드** 유지 (JPA 관리 객체에 적합) |

## 실행

```text
cd member-board-patterns2
.\gradlew.bat bootRun
```
