# RoomFit Social

`roomfit` 원본에 **Google · Naver · Kakao** 소셜 로그인을 추가한 프로젝트입니다.

## 실행

```powershell
.\gradlew.bat bootRun
```

- **http://localhost:8083** (MySQL, DB: `roomfit_social`)
- dev(H2): http://localhost:8084
