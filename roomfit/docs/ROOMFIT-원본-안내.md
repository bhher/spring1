# roomfit (원본)

이 폴더는 **소셜 로그인 없이** 폼 로그인(아이디/비밀번호)만 사용하는 버전입니다.

## 소셜 로그인 버전

Google · Naver · Kakao OAuth2가 포함된 프로젝트:

- **경로:** `d:\spring1\roomfit-social`
- **변경 목록:** [roomfit-social/docs/ROOMFIT-SOCIAL-변경사항.md](../roomfit-social/docs/ROOMFIT-SOCIAL-변경사항.md)

## 실행

```powershell
cd d:\spring1\roomfit
.\gradlew.bat bootRun
```

→ http://localhost:8081 (MySQL 프로필 기본)
